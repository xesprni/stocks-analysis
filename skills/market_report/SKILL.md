# 市场报告 Skill

## 分析框架

1. 宏观环境：总结主要经济体货币政策、利率、通胀趋势。
2. 市场指数：对比主要指数（S&P 500、纳斯达克、恒生、上证等）近期表现。
3. 资金流向：关注 ETF 资金流入/流出、行业轮动信号。
4. 市场情绪：通过新闻情感分析、VIX 或波动率指标判断整体情绪倾向。
5. 风险提示：识别潜在的系统性风险事件（地缘政治、政策突变等）。

## 输出要求

- 保持事实与观点分离，标注数据来源。
- 提供基准/乐观/悲观三种情景假设。
- 结论控制在 3-6 条，每条附证据引用。
- 使用中文输出。
