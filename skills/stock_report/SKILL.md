# 个股报告 Skill

## 分析框架

1. 基本面：营收、净利润、现金流、PE/PB 估值与行业对比。
2. 技术面：趋势（MA/MACD/布林带）、动量（RSI/KDJ）、量价关系、支撑/压力位。
3. 风险评估：识别关键风险因素及触发条件。
4. 操作建议：仓位建议、止损止盈位、入场区间。

## 输出要求

- 所有数据必须来自工具返回，禁止编造数字。
- 保持事实与观点分离，标注数据来源。
- 提供基准/乐观/悲观三种情景假设。
- 结论控制在 3-6 条，每条附证据引用。
- 使用中文输出。
- 数据收集完毕后立即输出最终 JSON 结论，不要再重复调用工具。
